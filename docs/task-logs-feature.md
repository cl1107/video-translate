# 任务日志功能说明

## 功能概述

现在视频翻译助手支持详细的任务处理日志显示，用户可以实时查看视频处理的每个步骤和进度。

## 新增功能

### 1. 实时日志记录

系统会记录任务处理过程中的所有关键步骤：

- ✅ **任务创建** - 文件验证、视频信息获取
- ✅ **服务检查** - Whisper 和 Ollama 可用性验证
- ✅ **音频提取** - FFmpeg 音频提取进度
- ✅ **音频切分** - 音频段落分割信息
- ✅ **语音识别** - Whisper 转录进度和结果
- ✅ **文本翻译** - Ollama 翻译进度和结果
- ✅ **字幕生成** - 最终字幕文件生成
- ❌ **错误处理** - 详细的错误信息和堆栈

### 2. 日志分类

日志按重要性分为四个级别：

- 🔵 **信息 (info)** - 一般处理步骤和进度更新
- 🟢 **成功 (success)** - 步骤完成确认
- 🟡 **警告 (warn)** - 非致命问题提醒
- 🔴 **错误 (error)** - 失败原因和错误详情

### 3. 用户界面改进

#### 任务列表增强

- 添加了展开/折叠按钮（上下箭头）
- 点击可查看任务详细信息和日志
- 保持简洁的卡片设计

#### 日志显示组件

- 时间戳精确到毫秒
- 彩色图标区分日志级别
- 支持长文本自动换行
- 最大高度限制，超出可滚动

## 使用方法

### 查看任务日志

1. **上传视频文件**

   - 拖拽或选择视频文件上传
   - 系统自动创建翻译任务

2. **监控处理进度**

   - 切换到"任务列表"标签页
   - 查看任务状态和进度条

3. **查看详细日志**
   - 点击任务卡片右侧的下箭头 ⬇️
   - 展开后可看到完整的处理日志
   - 日志按时间顺序显示，最新的在下方

## 技术实现

### 数据库存储

- 新增 `task_logs` 表存储日志记录
- 支持日志的增删查操作
- 自动清理过期日志

### 实时更新

- 使用 IPC 通信机制
- 主进程记录日志，渲染进程实时显示
- 无需手动刷新，自动更新

### 性能优化

- 日志组件懒加载
- 虚拟滚动支持大量日志
- 定期清理临时日志文件

## 故障排除

### 日志不显示

1. 检查任务是否已展开
2. 确认网络连接正常
3. 重启应用重新加载

### 日志更新延迟

1. 这是正常现象，某些步骤需要时间
2. 长时间无更新可能表示处理卡住
3. 可以尝试暂停后重新开始任务
